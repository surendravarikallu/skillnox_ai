1. **Start Interview**:
   - User selects interview type
   - AI generates questions
   - AI interviewer avatar appears

2. **During Interview**:
   - AI interviewer visible (animated avatar)
   - Questions displayed
   - User can type or speak answers
   - Real-time emotion analysis
   - Voice clarity tracking

3. **Answer Submission**:
   - AI evaluates answer
   - Provides score and feedback
   - Moves to next question

4. **Completion**:
   - Overall scores calculated
   - Detailed feedback provided
   - Results saved

## 📋 **HOW TO USE**

### Start the System:

1. **Start Python AI Service**:
   ```bash
   cd python-ai
   python services/api_service.py
   ```
   Should see: "Using fine-tuned LLM model"

2. **Start Node.js Backend**:
   ```bash
   npm run dev
   ```

3. **Access Application**:
   - Open: `http://localhost:5000`
   - Register/Login
   - Start Interview

### Test Interview:

1. Click "Start Interview"
2. Select interview type (Technical, HR, etc.)
3. See AI interviewer avatar
4. Answer questions (type or speak)
5. Submit answers
6. View results
