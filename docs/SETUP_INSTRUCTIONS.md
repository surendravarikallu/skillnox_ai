## Setup Instructions for Interview System

Moved here from `SETUP_INSTRUCTIONS.md` at the project root.

High-level steps:
- Start Python AI: `cd python-ai && python services/api_service.py` (check `/health`).
- Start Node backend: `npm run dev` (serves on port 5000).
- Open `http://localhost:5000`, log in with test accounts, and verify AI features (questions, avatar, voice, evaluation).
