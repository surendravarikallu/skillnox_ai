## Quick Start

1. Create/update `.env` with `DATABASE_URL`, `JWT_SECRET`, `PYTHON_AI_SERVICE_URL`, and `SESSION_SECRET`.  
2. Run `npm run db:push` then `npm run seed`.  
3. Start Python AI (`cd python-ai && python services/api_service.py`) and Node backend (`npm run dev`).  
4. Open `http://localhost:5000` and log in with the seeded admin or student account.
